
# Next.js OpenJira - Tasks App Docker
```
docker-compose up -d
```
## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
MongoDB URL local:
```
mongodb://localhost:27017/jira
```

## Config Environment Variables
Rename .env.template / .env

## Fill test database

http://localhost:3000/api/seed
